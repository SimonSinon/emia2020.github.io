# Ai-mate - 3-in-1 AI Platform for Animators

**By Flying Marshmallow Team**

## Overview

Ai-mate is a comprehensive AI-powered platform designed to revolutionize animation workflows. This prototype demonstrates a 3-in-1 solution that empowers animators with cutting-edge AI tools to enhance productivity and creative output.

## Features

### 1. AI-nimate - Animation Enhancement Tool
- **Frame-Interpolation**: One-click increase from 24/30 fps to 60 fps using AI frame interpolation for smoother motion
- **Super-Resolution Upscaling**: Enhance resolution of existing animations with AI-generated pixels for sharper, more detailed visuals

### 2. SyncYourVoice - Automated Lip-Sync & Voice Matching
- Automatically fuses voice and animation clips into seamless, natural-looking performances
- Integrates with common 2D and 3D software
- Context-aware facial expressions and head movements that reflect tone, emotion, and emphasis in dialogue

### 3. ProtoAnimate - AI-Driven Prototyping
- **Rapid Idea Generation**: Analyzes sketches, descriptions, or references to generate preliminary motion sequences and scene layouts
- **Original Content Creation**: Ensures ethical practices by generating content based on artists' past works and unlicensed stock images

## Usage

This is a presentation prototype designed to demonstrate the user experience and workflow of the Ai-mate platform. Open `index.html` in a web browser to start the demo.

### Demo Account Credentials
- **Email:** `demo@ai-mate.com`
- **Password:** `demo123`

### Navigation Flow
1. Start at landing page → Learn about the 3 AI tools
2. Click "Get Started" or "Sign In" → Access login/register pages
3. Login with demo credentials (or create a new account)
4. Upon successful login → Redirected to workspace dashboard
5. View AI tools quick access cards
6. Create or open projects
7. Experience the editor workspace with AI tools integration

### Authentication Features
- **Login Page** (`login.html`) - Sign in with email and password
- **Register Page** (`register.html`) - Create a new account
- **Protected Routes** - Projects and editor pages require authentication
- **Session Management** - User sessions persist during browsing
- **Logout Functionality** - Available in projects dashboard and editor
